# VjestackaInteligencija

This repository contains lab exercises and assignments for the Artificial Intelligence course.

The project covers important machine learning and data analysis topics, including:
- data preprocessing
- handling missing values
- encoding categorical data
- normalization and standardization
- classification
- regression
- clustering
- model training and testing
- model evaluation
- data visualization

The work is mainly implemented in Jupyter Notebook and includes practical examples, experiments and course-related tasks.
